# 🚀 Pro Job Portal (Frontend Project)

A colorful and modern **Job Portal Web Application** inspired by platforms like Naukri.
This project is built using **HTML, CSS, and JavaScript** with LocalStorage for data handling.

## 🌟 Features

* 🎨 **Colorful UI** with different backgrounds for each page
* 🔐 **User Authentication** (Sign Up & Login)
* 💼 **Job Listings** (12 jobs displayed)
* 🔍 **Search & Filter Jobs**
* 📄 **Apply for Jobs**
* 📊 **Application Tracker**
* 👤 **Profile Management** (10+ fields)
* ⭐ **Rating System**
* 🌙 **Dark / Light Mode**
* 💾 **LocalStorage Integration** (data persistence)

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in any browser
3. Start using the portal

---

## 💡 How It Works

* User registers and logs in
* Jobs are displayed dynamically
* Users can apply for jobs
* Applications are stored in LocalStorage
* Tracker shows applied jobs
* Profile & rating data are saved locally
